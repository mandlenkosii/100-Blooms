# 100-Blooms
Website
This is a website for a flower and gift shop that specilizes in roses and many other things.
## LANDING PAGE
The is a landing page that you get to when you open the link to the website.
It has sections that cover firstly the navigation bar that has links to the home page, services, gallery and the contact button
The home page has sections that shows the services, one that shows the background story, reviews from previous costumers,a sections where you can contact the shop and lastly a footer.

##OTHER PAGES 
There is a page where you can browse the gallery of the shop

A page where you can shop the specified flowers that one would want.
